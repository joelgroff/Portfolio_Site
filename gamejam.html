USF GAME JAM

DIRECTIONS
1. Let game finish loading in your browser.
2. Click the game.
3. Press space to start.
4. Use arrows to move.

GOAL
Build a fun game within the time limit!

OVERVIEW
The USF Game Development Club hosted a game jam in fall of 2016. Participants had eight hours to finish their game and were judged based on UI design, sound, fun, and polish.

Unfortunately, due to work and travel I was only able to participate in the last hour of the jam. My final game, where players control a square struggling to survive an onslaught of other angry squares, was entirely created within that hour.

TOOLS
Unity
C#

EXPERIENCE
Coding as fast as possible for one hour
