USF 2016 Game Jam

DIRECTIONS
1. Let game finish loading in your browser
2. Click game
3. Press space to start
4. Use arrows to move

GOAL
Build a fun game in an hour!

OVERVIEW
The USF Game Development Club hosted a game jam in fall of 2016. Participants had eight hours to finish their game and were judged based on UI design, sound, fun, and polish.

Unfortunately, due to work and travel I was only able to participate in the last hour of the jam. My final game, where players control a square struggling to survive an onslaught of other squares, was entirely created within that hour.

TOOLS
Unity
C#

EXPERIENCE
Coding as fast as possible for one hour
