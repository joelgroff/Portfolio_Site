THE TREE OF OOT

GOAL
Build a simple, complete game from scratch.

OVERVIEW
My first game, The Tree of Oot, is a 2D endless-dodger, where the player controls Oot, a flying squirrel. Oot can open his wings to slow his fall and better dodge pinecones. Each successfully dodged pinecone increases the player's score. Video coming soon!

TOOLS
C++
SDL
Photoshop

EXPERIENCE
2D collision detection
User input control
Game loop management
Random object generation
Importing art and linking to objects
